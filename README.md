# Kalkulator Napiwków i Podziału Rachunku

Nowoczesna aplikacja webowa do obliczania napiwków i dzielenia rachunku między osoby.

## Funkcje

- Wprowadzanie kwoty rachunku
- Suwak napiwku (0%–30%, domyślnie 10%)
- Wybór liczby osób (minimum 1)
- Podsumowanie: napiwek, suma całkowita, kwota na osobę
- Przeliczanie na żywo podczas edycji

## Technologie

- React
- Tailwind CSS
- Vite

## Uruchomienie

```bash
npm install
npm run dev
```

## Budowanie

```bash
npm run build
```
